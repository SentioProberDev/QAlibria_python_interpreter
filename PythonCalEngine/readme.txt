Requirement
----------------

# IDE
    Install Python engine https://www.python.org/downloads/
    Install Visual Studio Code https://code.visualstudio.com/download

# Open PythonCalEngine by Visual Studio Code(VS Code) IDE
    Click "Ctrl+Shift+P" from VS Code IDE
    Select "Python: Create Environment" to install python package

# Install Package:
    Click "Ctrl+Shift+P" from VS Code IDE
    Select "Python: Create Terminal"
    Type "pip install scikit-rf" in terminal to install scikit-rf package
    Type "pip install matplotlib" in terminal to install scikit-rf package
	Type "pip install metas_unclib" in terminal to install scikit-rf package
